# vantage-voice

voice agent

## Quick Start

1. **Edit your handler:**
   Open `handler.py` and customize the event handlers for your use case.

2. **Push your code to the server:**
   ```bash
   vr push
   ```

3. **Open the web UI:**
   ```bash
   vr open
   ```

From here you can start creating environments, testing your agent in the debugger, and more.

## Project Structure

```
vantage-voice/
├── handler.py              # Main agent code (entry point)
├── README.md               # Project documentation
├── requirements.txt        # Python dependencies
└── .voicerun/              # Configuration directory
    ├── agent.yaml          # Agent metadata and configuration
```
